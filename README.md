# Weather CLI App

A command-line weather app built with Python that fetches real-time data.

## Features
- Search weather by city name
- Shows temperature, feels like, humidity and conditions
- Uses OpenWeatherMap API

## How to run
```bash
pip3 install requests
python3 weather.py
```

## What I learned
- Making API calls with the requests library
- Parsing JSON responses
- Working with real external data
